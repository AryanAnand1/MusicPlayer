# MusicPlayer
Music Player with a Graphical User Interface developed in Ruby

Two albums included in the repository, to listen to any other album, just replace the songs in album1, or album2 folders

How to Run:
1. Will require ruby installed on the device, as well as gosu library.
2. To run, on command prompt navigate to the file directory and then to launch the application write ruby source_code.rb
3. file location ruby source_code.rb
